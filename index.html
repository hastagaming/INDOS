Mengenal INDOS

INDOS (Indonesia OS) adalah sistem operasi minimalis yang berjalan di dalam Termux tanpa memerlukan akses root. Dikembangkan untuk performa maksimal di perangkat Android.

⚡ Super Ringan

Hanya menggunakan biner inti BusyBox asli, menjaga penggunaan RAM tetap rendah di HP Realme kamu.

🛠️ Standalone

Tidak memerlukan file distro besar lainnya. Semua yang kamu butuhkan sudah ada di dalam repo ini.

Instalasi

Salin dan tempel perintah berikut di Termux:

git clone https://github.com/hastagaming/INDOS && cd INDOS && ./install.sh

Memulai Sistem

Setelah script instalasi selesai, kamu bisa masuk ke lingkungan INDOS kapan saja dengan mengetik:

indos
